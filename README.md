# Selection Translator for Chromium

Моё расширение показывает компактную кнопку рядом с выделенным текстом и по клику открывает pop-up с переводом.
Работает сильно быстрое других переводов, особенно в сравнении с deepL, работает чисто на странице не открывая сайт переводчика.

## Что умеет

- отслеживает выделение текста на любой странице;
- показывает компактную кнопку рядом с выделением;
- открывает pop-up с переводом по клику на кнопку или через контекстное меню;
- переводит текст с автоопределением исходного языка;
- позволяет копировать результат;
- поддерживает выбор целевого языка через страницу настроек.

## Как установить

1. Откройте `chrome://extensions/`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked`.

## Как использовать

1. Выделите текст на странице.
2. Нажмите на маленькую кнопку перевода рядом с выделением.
3. При необходимости откройте настройки расширения и смените целевой язык.

## Техническая заметка

Перевод выполняется через запрос к `translate.googleapis.com` из background service worker.
